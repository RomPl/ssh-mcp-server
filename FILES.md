# Обзор файлов проекта

## Основные файлы

| Файл | Назначение |
|------|------------|
| `server_http.py` | HTTP/SSE сервер + OAuth + REST bridge |
| `server_stdio.py` | stdio сервер |
| `auth_oauth.py` | OAuth 2.1, JWT, коды авторизации, валидация |
| `tools.py` | Список MCP tools и `dispatch()` |
| `ssh_client.py` | SSH/SFTP клиент и пул соединений |

## Документация

| Файл | Назначение |
|------|------------|
| `README.md` | Общее описание |
| `QUICKSTART.md` | Быстрый старт |
| `SECURITY.md` | Политика безопасности |
| `CHANGELOG.md` | Изменения по версиям |
| `CONTRIBUTING.md` | Правила внесения изменений |

## Конфигурация

| Файл | Назначение |
|------|------------|
| `.env.example` | Шаблон настроек |
| `.env` | Реальные секреты (не коммитить) |

## Важно

Не коммитить:
- `.env`
- любые backup‑файлы `*.bak_*`
